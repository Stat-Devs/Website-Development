Training

The vision for providing workshop training is to equip the organizations with software tools, enrich their Business Intelligence, Data Science and Research teams with modern day software practices and tools. We currently offer onsite and remote training to employees and university students across all backgrounds in following courses.

Business Data Analysis in R

The course is designed for Business Professionals to learn the fundamentals of Data Science in Business Analytics context. The course will teach fundamentals of Data Analysis, Visualization using R and Tidy-verse packages.

Time Series Analysis & Forecasting

The course teaches the fundamentals of Time Series Analysis and explains why certain trends in business data occur seasonally. It utlizes Timetk and tidymodels packages in R to fit and forecast time series models.

Statistics & Machine Learning in R

The course will utilize the tidy models packages to extract raw data and prepare data pipelines with Feature Engineering and Hyper parameter tuning. It also teaches the fundamentals of deep learning focus on business use cases in Machine learning.

Spatial Data Analysis

The course teaches the classes and methods of spatial data and how to visualize spatial data using plot, map and scale concepts. The course uses sp package for spatial objects and sf package for Simple features in Spatial Data.

Shiny Applications

The course teaches Shiny which is the new shift in Data Analytics to provide interactive dashboards and advance analytics. We teach key concepts of software engineering with dashboard principles and reactivity to present your data findings in interactive manner.

R for Data Science

The courses teaches fundamentals of Data Science and Machine Learning using R. It uses tidyverse package for data wrangling, analyzing and visualizing data and teaches concepts to build robust models and deploy them through R.

Reporting & Presentation in Quarto

The course explores the techniques to use Quarto for scientific and technical publishing high-quality articles, reports, presentations, websites, blogs, and books in HTML, PDF, MS Word, ePub, and more.

R in Production

The course explores the use of R for production applications. It uses software engineering principles to organize your shiny app in a way that is time and code efficient using renv, plumber and Docker.

Reproducible Workflows

Reproducibility in Research is also known as replicability and repeatability, is a major principle underpinning the scientific method. The course teaches the fundamentals of reproducible workflows and help you doing quality research which can be verified, repeated, and reproduced.

R Package Development

The course teaches the fundamentals of R Package which can be used to replicate the data analysis and code in R. We explore basics of Package development, testing, documentation and Maintenance of your R packages.
